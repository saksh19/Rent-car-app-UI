#Rent Car App - UI
##Introduction

The Rent Car App is a modern and responsive Flutter-based mobile UI designed to provide users with a smooth and intuitive car-rental browsing experience. The app features a clean layout for viewing available cars, filtering options, and a detailed car preview screen with specifications, pricing, and images. Built with reusable components, animations, and scalable architecture, the UI is optimized for performance and easy future integration with backend APIs.
